# EyeVideo

这是一款使用Flutter开发的高仿开眼视频的短视频APP

## 主要使用的技术点

* Dart扩展函数的使用

* Dio网络库优雅封装以及使用

* Bloc状态管理以及实现UI和逻辑分离

* json_serializable的使用

* ScreenRuler Flutter UI尺寸和字体大小的统一适配

* 列表刷新和加载更多组件实现

## 目前支持的平台有

* Android

* iOS

* Mac Desktop

* Linux Desktop

* Web Chrome

* Web Server
